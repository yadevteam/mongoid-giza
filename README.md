# Mongoid::Giza

Mongoid layer for the Sphinx fulltext search server that supports block fields and dynamic indexes

## Installation

Add this line to your application's Gemfile:

    gem 'mongoid-giza'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install mongoid-giza
